# Cell Stage

A Spore-inspired idle evolution game. Grow your cell, collect nutrients, evolve upgrades, and explore biomes in a 2D arena.

## Stack

- Next.js 16 + React 19
- Tailwind CSS v4
- Canvas-based arena renderer
- Client-side state with localStorage persistence

## Run locally

```
npm install
npm run dev
```

## How to play

- **Explore** biomes to send your cell into a 2D arena where it auto-hunts for food, fights rivals, and collects loot
- **Evolve** upgrades in the Evolution Lab to boost toxicity, membrane, motility, and adaptation
- **Collect** nutrients, biomass, and mutation fragments from arena runs to fund your evolution
- Idle nutrient generation continues while you're away
